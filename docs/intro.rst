Introduction to PyZEAL's Documentation
======================================

This is the introduction to the numerical project *PyZEAL* that provides facilities for the calculation of zeros and poles of holomorphic and meromorphic functions.
At the moment, there is no information here apart from an example citation :cite:p:`kravanja.1999` and an example mathematical formula:

.. math::
   :nowrap:

   \begin{equation}
   \begin{split}
   a &= 1, \quad b = 2 \\
   y &= \int_a^b f(x) ~\mathrm{d}x
   \end{split}
   \end{equation}
